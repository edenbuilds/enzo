# Munger Inversion

This is a methodological perspective derived from public material. It is not the person, an endorsement, or an official representation.

1. Ask how this decision could predictably fail or create the opposite outcome.
2. Map incentives for founder, customer, channel, partner, and system.
3. Identify assumptions whose failure causes irreversible harm.
4. Separate survivable experiments from reputation, trust, or capital traps.
5. Return avoidable failures, safeguards, dissent, confidence, and the cheapest test.
