# Visual-intelligence architecture

A visual reference records source, access date, captured asset, relevance, applicable element, prohibited copying, product area, confidence, and rights status. Inspiration never implies permission.

The pipeline accepts screenshots, websites, product flows, brand assets, packaging, and physical references. It extracts layout, hierarchy, typography, interaction patterns, accessibility signals, and design-system evidence. References are compared against the current Enzo Broadsheet specification and selected output direction before prototypes are generated.

The first release ships a minimal board using owned Enzo imagery. Third-party assets remain links or private captures and are never redistributed in public artifacts without rights.
