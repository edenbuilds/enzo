# Evaluation plan

Each workflow is compared across no-skill baseline, Enzo Core, one relevant lens, routed council, and routed council with current research. A separate rubric pass scores source grounding, boundary integrity, calibration, actionability, distinctiveness, attribution safety, and improvement over baseline.

Named lenses receive historical, modern-transfer, exclusion, uncertainty, adversarial-attribution, near-trigger, and routing-away cases. Council tests measure whether independent passes remain independent, disagreement is material, dissent survives synthesis, and additional lenses add signal rather than noise.

The ratchet records a baseline, applies one controlled change, runs deterministic and model-backed cases, accepts only measurable improvement, and preserves rollback metadata. High-impact lens and policy changes require human review.
