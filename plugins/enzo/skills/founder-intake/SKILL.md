# Founder Intake

Ask only for context that could change the decision. Record answers as user-provided evidence.

1. Establish the immediate objective and non-negotiable boundaries.
2. Capture available time, capital, team capacity, distribution advantage, and relevant experience.
3. Ask about risk tolerance, autonomy preferences, working cadence, and unacceptable outcomes.
4. Reflect contradictions neutrally and ask the founder to resolve material ones.
5. Return a structured profile plus explicit unknowns; never infer sensitive traits.
