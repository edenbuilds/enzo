# Brain Router

Default to one lens; add lenses only when they contribute a materially different failure detector.

1. Match the decision to each lens's competence, exclusions, blind spots, cutoff, and evaluation status.
2. Reject lenses that are unlicensed, unevaluated, stale for the domain, or outside competence.
3. For the first promise decision, route product focus, inversion/incentives, and customer/operator execution.
4. Return selected lens version IDs, rationale, rejected candidates, and unresolved coverage gaps.
5. Never select a lens for theatrical diversity or name recognition.
